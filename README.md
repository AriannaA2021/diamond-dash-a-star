# Diamond Dash

A grid-based chase game where you race to collect diamonds while avoiding an intelligent opponent that tracks you using the A* pathfinding algorithm.

## How to Play

- **Objective:** Collect all 3 **Blue Diamonds** and reach the **Yellow Exit** before time runs out.
- **Avoid:** The **Red Opponent** chases you every time you move. If it catches you, the game is over.
- **Win:** Gather all diamonds and step on the exit tile.

### Controls
- **Arrow Keys:** Move Up, Down, Left, or Right.

## Setup & Installation

### 1. Install Python
Ensure you have Python installed on your system.

### 2. Install Dependencies
This game requires the `pygame` library. You can install it directly via your terminal:

```bash
pip install pygame
```

## How to Run
Run the game script using Python:

```bash
python "diamondDash.py"
```

*(Note: You may want to rename the python file to something simpler like `game.py` for easier running!)*
